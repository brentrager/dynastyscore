Dynasty Score By Brent Rager

Dynasty Score is all about providing information on how the teams in your league compare. For some, it's about knowing where they need to improve. For others, it's about knowing their team is good, but finally having a way to prove it to everybody else in the league.

Rankings By FantasyPros.com

We compile a set of rankings using your dynasty league teams and the consensus rankings provided by FantasyPros.com.

Redraft Starter Rankings

We find the consensus redraft ranking of each player on each team and build the best set of starters based on the league starting configuration.

Dynasty Starter Rankings

We find the consensus dynasty ranking of each player on each team and build the best set of starters based on the league starting configuration.

Combined Starter Rankings

We add up the consensus redraft and dynasty ranking of each player on each team and build the best set of starters based on the league starting configuration.

Redraft Depth Rankings

We find the consensus redraft ranking of each player on each team and build the best set of starters plus one depthspot at each of the WR, RB, TE, and Flex positions based on the league starting configuration.

Dynasty Depth Rankings

We find the consensus dynasty ranking of each player on each team and build the best set of starters plus one depthspot at each of the WR, RB, TE, and Flex positions based on the league starting configuration.

Combined Depth Rankings

We add up the consensus redraft and dynasty ranking of each player on each team and build the best set of starters plus one depthspot at each of the WR, RB, TE, and Flex positions based on the league starting configuration.
